# Resume / Candidate Screening System

## Project Overview

This project is an AI-powered Resume Screening System that automatically analyzes and ranks candidate resumes based on their relevance to a given job description.

The system uses Natural Language Processing (NLP) and Machine Learning techniques to compare resumes with job requirements and identify the most suitable candidates.

---

## Objective

To automate the resume screening process and help recruiters quickly identify the best candidates for a specific job role.

✔ Recruiter-friendly candidate selection process

---

## Project Workflow

1. Upload candidate resumes in PDF format
2. Extract text from each resume
3. Define the job description
4. Convert text into TF-IDF vectors
5. Calculate similarity scores using Cosine Similarity
6. Rank candidates according to match score
7. Display results using charts and visualizations
